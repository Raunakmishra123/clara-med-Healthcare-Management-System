# ClaraMed - Healthcare & Doctor Appointment Management System

ClaraMed is a modern, user-friendly web application designed to simplify booking and managing doctor appointments. It connects patients with medical specialists, making healthcare access convenient and efficient.

## Key Features

- **Specialist Directory**: Browse and filter doctors by medical specialties (e.g., General Physicians, Gynecologists, Dermatologists, Pediatricians, Neurologists, Gastroenterologists).
- **Easy Booking**: View doctor availability and book appointments for specific dates and time slots.
- **My Appointments**: Keep track of scheduled appointments, with options to pay online or cancel visits if needed.
- **User Profiles**: Manage personal information, contact details, and upload profile pictures.
- **Secure Payments**: Integrated payment options (Stripe & Razorpay) for paying consultation fees online.

## Technology Stack

- **Frontend Framework**: React.js with Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Notifications**: React Toastify
- **State Management**: React Context API

## How to Run Locally

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Raunakmishra123/clara-med-Healthcare-Management-System.git
   cd clara-med-Healthcare-Management-System
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
